Sources & verification

All radio specifications used in this calculator are taken from the manufacturer datasheets cited below. To verify any number, open the linked PDF and find the referenced table.

How verification works

Each TX current, RX current, and sensitivity value in this calculator carries a citation tag like [1] linking to the specific datasheet. The numbers are read directly from the manufacturer's electrical-characteristics tables — not estimated, not interpolated.

In the leaf and hub tabs, the radio spec panel shows every parameter alongside the citation, so you can see at a glance what data the calculation is using and where it came from.

Bibliography

    How to independently verify a number

    1. Pick the radio you want to verify in the Leaf or Hub tab.
    2. Note the TX current shown in the radio spec card (e.g. "44.5 mA at +13 dBm").
    3. Open the datasheet link from the bibliography below.
    4. Navigate to the electrical characteristics table — usually section 4 or 5 in the datasheet.
    5. Compare the typical value at the matching output power. They should match within ±10% (datasheet typ. values vary slightly across revisions and operating conditions).

    Where measured vs typ. values may differ

    Datasheet "typical" values assume specific test conditions (usually 3.3 V supply, 25 °C, class-E match, etc.). Real applications may differ:

    • PCB matching network losses: +5–15% TX current vs typ. without optimal matching
    • Antenna mismatch: Effective TX power can be 2–4 dB below set point
    • Temperature: Current can rise 10–20% at extreme operating temperatures
    • Crystal startup transient: First ~200 µs of TX draws more current than the listed steady-state value

    For a final production design, validate against bench measurements on the actual hardware before committing to battery sizing.
